⚠️ Principais limitações

⏱️ 1. Dados não são totalmente em tempo real
Muitos ativos têm atraso (ex: 15 minutos)
Em mercados internacionais isso varia bastante
👉 Problema: ruim para day trade ou decisões rápidas

🔌 2. API não oficial
A biblioteca yfinance não é mantida pelo Yahoo oficialmente
Pode quebrar a qualquer momento se o site mudar
👉 Isso já aconteceu várias vezes

📉 3. Qualidade dos dados pode falhar
Dados históricos podem ter:
Buracos (missing data)
Ajustes inconsistentes (split/dividendos)
Indicadores fundamentais às vezes vêm incompletos

🌎 4. Cobertura irregular (principalmente no Brasil)
Para ativos da B3 (como Petrobras):
Nem todos os dados estão disponíveis
Menos detalhes que ações dos EUA (como Apple Inc.)

📊 5. Dados fundamentais limitados
Não é tão profundo quanto plataformas profissionais
Algumas métricas:
podem estar desatualizadas
podem variar de fonte para fonte   

⚡ 6. Sem execução de ordens
Não dá pra comprar/vender ativos
Não é uma corretora
👉 Só análise e acompanhamento

🚫 7. Limites implícitos de uso
Muitas requisições podem gerar bloqueio temporário
Não é feito para sistemas pesados ou produção

🔒 8. Sem garantias (uso não profissional)
Não há SLA (garantia de disponibilidade)
Não é recomendado para:
sistemas financeiros críticos
trading automatizado real
